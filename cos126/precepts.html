Precepts

The worksheets and code on this page will largely be covered in precept, whereas the recommended exercises and websheets are here for you to practice your skills at home. The websheets are programming exercises you can type and submit in your browser that give you instant feedback and have reference solutions. We've also created a tool called the Java Visualizer that will animate the execution of any code you like, to see exactly what Java does each step of the way.
