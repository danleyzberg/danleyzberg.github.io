Precepts

The worksheets and code on this page will largely be covered in precept, whereas the recommended exercises and websheets are here for you to practice your skills at home. Websheets are online programming exercises you can write and submit in your browser which give you instant feedback and have reference solutions if you get stuck. We've also created a tool called the Java Visualizer that will animate the execution of any code, showing you exactly what Java does each step of the way.
